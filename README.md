# AGROSENSE-AI
🌱 AgroSense AI
AI-Driven Crop Disease Detection & Advisory Platform
📌 Overview

AgroSense AI is an intelligent agriculture system that uses Artificial Intelligence and Deep Learning to detect crop diseases and provide actionable recommendations. The platform enables farmers to upload crop leaf images and receive accurate disease diagnosis along with treatment and prevention suggestions.

This project aims to bridge the gap between traditional farming and modern AI-based precision agriculture by delivering real-time decision support through a user-friendly web application.

🚀 Features
🌿 Crop Disease Detection using CNN models
📊 Environmental Risk Prediction based on conditions like temperature & humidity
💊 Treatment & Prevention Suggestions (chemical + organic remedies)
🌐 User-Friendly Web Application (HTML, CSS, JS + Flask)
⚡ Real-Time Prediction System
🤖 AI Chatbot (Gemini Integration) for farming guidance
🌾 Multi-Crop Support (Rice, Wheat, Tomato, Cotton, etc.)
🧠 AI Models Used

Convolutional Neural Networks (CNN)
Transfer Learning Techniques
ResNet50 (primary model)
EfficientNet & MobileNet (comparative study)

📈 Achieved high accuracy (~96%) on PlantVillage dataset.

📂 System Architecture
User uploads crop leaf image
Image is preprocessed (resize, normalize)
CNN model predicts disease
Output mapped to disease class
Advisory system provides treatment & prevention
Results displayed on web interface
