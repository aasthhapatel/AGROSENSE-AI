# 🌱 AgroSense AI  
## AI-Driven Crop Disease Detection & Advisory Platform  

---

## 📌 Overview  
AgroSense AI is an intelligent agriculture system that uses Artificial Intelligence and Deep Learning to detect crop diseases and provide actionable recommendations.  

This project enables farmers to upload crop leaf images and receive accurate disease diagnosis along with treatment and prevention suggestions. It bridges the gap between traditional farming and modern AI-based precision agriculture by delivering real-time decision support through a user-friendly web application.

---

## 🚀 Features  
- 🌿 Crop Disease Detection using CNN models  
- 📊 Environmental Risk Prediction using temperature & humidity  
- 💊 Treatment & Prevention Suggestions  
- 🌐 User-Friendly Web Application (HTML, CSS, JS + Flask)  
- ⚡ Real-Time Prediction System  
- 🤖 AI Chatbot (Gemini Integration)  
- 🌾 Multi-Crop Support  

---

## 🧠 AI Models Used  
- Convolutional Neural Networks (CNN)  
- Transfer Learning  
- ResNet50 (Primary Model)  
- EfficientNet & MobileNet  

📈 Achieved ~96% accuracy on PlantVillage dataset
